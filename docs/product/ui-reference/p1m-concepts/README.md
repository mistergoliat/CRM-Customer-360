# P1M Concepts — PesasChile HUB

Estas imágenes son las referencias visuales canónicas para la evolución UI-first de P1M.

## Regla de interpretación

- Imagen de listado: entrada principal del módulo.
- Imagen de detalle: vista interna al seleccionar una entidad.
- Panel lateral: resumen contextual, no módulo independiente.
- Copilot: asistente contextual, no dominio principal.
- Tabs internas: subvistas del módulo, no elementos del sidebar.
